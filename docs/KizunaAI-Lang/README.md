## 実行

```
gem install r-fxxk
cd docs/KizunaAI-Lang
./kizunaai helloworld.kizunaai
```

## キズナアイ言語

| 構文 | 言語仕様 |
|--|--|
| はいどうもー! | ポインタを右へ移動 |
| 最高かよ! | ポインタの指す値を1増やす |
| ふぁっきゅー | ポインタを左へ移動 |
| アイた〜んビーム!! | ポインタの指す値を1減らす |
| お・ね・が・い | ポインタの指す値が0なら 対応する '完璧ですね!' までジャンプ |
| 完璧ですね! | ポインタの指す値が0で無いなら 対応する 'お・ね・が・い' までジャンプ |
| 危機回避〜! | ポインタの指す値を出力する |
| ちょちょいのちょいですよ | 入力から1バイト読み込む|
